# Subrat Portfolio

A modern, responsive portfolio website built with React and Vite, showcasing projects, skills, and professional experience.

## 🚀 Live Demo

**View the live portfolio:** [https://subrat-portfollio.vercel.app/](https://subrat-portfollio.vercel.app/)

## 📸 Preview

[![Portfolio Preview](/screenshots/homepage-1.jpeg)](https://subrat-portfollio.vercel.app/)

## ✨ Features

- **Responsive Design**: Optimized for all devices (desktop, tablet, mobile)
- **Modern UI**: Built with React Bootstrap and custom styling
- **Interactive Elements**:
  - Particle animations with tsparticles
  - Parallax tilt effects
  - Typewriter effect for dynamic text
  - GitHub contribution calendar
- **Smooth Navigation**: React Router for seamless page transitions
- **PDF Resume View**: Integrated PDF viewer for resume
- **Professional Sections**: Home, About, Projects, Skills, Experience, Contact

## 🛠️ Tech Stack

### Frontend

- **React 18** - UI framework
- **Vite** - Build tool and development server
- **React Bootstrap** - UI component library
- **React Router DOM** - Client-side routing
- **Axios** - HTTP client for API calls

### Animations & Effects

- **React TSParticles** - Particle animations
- **React Parallax Tilt** - 3D tilt effects
- **Typewriter Effect** - Dynamic text animation
- **React GitHub Calendar** - Contribution graph

### Icons & Assets

- **React Icons** - Icon library
- **Google Fonts** - Custom typography (Cutive Mono, Monoton)

## Installation

1. Clone the repository:

```bash
git clone https://github.com/yourusername/subrat-portfolio.git
cd subrat-portfolio
```

2. Install dependencies:

```bash
npm install
```

3. Start the development server:

```bash
npm run dev
```

4. Build for production:

```bash
npm run build
```

5. Preview production build:

```bash
npm run preview
```

## 🚀 Deployment

This portfolio is deployed on Vercel. To deploy your own version:

1. Push your code to GitHub
2. Connect your repository to Vercel
3. Vercel will automatically build and deploy your site

## 🎨 Customization

### Adding New Projects

1. Create project data in the appropriate component
2. Update the projects section with your project details
3. Add project images to the `public/assets` folder

### Modifying Styles

- Custom CSS can be added alongside React Bootstrap classes
- Update color schemes and layouts in component files

### Adding New Sections

1. Create new components in the `src/components` folder
2. Add routes in the main App component
3. Update navigation accordingly
